# Passgen

A simple cli util for generating passwords in a couple of clicks.

## One-line installation

### Via curl (macOS/Linux)

``` bash
curl -fsSL https://raw.githubusercontent.com/esavenko/passgen/master/install.sh | sudo bash
```

### Go installation

``` bash
go install github.com/esavenko/passgen/cmd/passgen@latest
```

## Possibilities

- Password generation
  - with/without digits
  - with/without special symbols
  - arbitrary length

## Requirements

- For manual installation
  - Go (1.24+)
